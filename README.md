# Waste Classification for Recycling

Проект выполнен в рамках производственной практики по дисциплине **«Технологии искусственного интеллекта»**.

Цель проекта — разработка системы классификации бытовых отходов по изображению с использованием методов глубокого обучения.

---

## Возможности

* классификация изображений отходов по 6 классам;
* сравнение пяти современных архитектур нейронных сетей;
* оценка качества моделей по нескольким метрикам;
* анализ ошибок классификации;
* демонстрационный веб-интерфейс на Streamlit.

---

## Классы отходов

Модель определяет следующие категории:

* Cardboard
* Glass
* Metal
* Paper
* Plastic
* Other

После классификации система отображает:

* определённый класс;
* вероятность предсказания;
* рекомендацию по дальнейшей сортировке.

---

## Используемые архитектуры

В работе были исследованы следующие модели:

* ResNet18
* DenseNet121
* MobileNetV3-Large
* EfficientNet-B0
* Vision Transformer (ViT-B-16)

Все модели обучались с использованием **Transfer Learning** на предобученных весах ImageNet.

## Используемый датасет

Проект использует открытый датасет:

**Recyclable and Household Waste Classification Dataset**

Особенности датасета:

* изображения бытовых отходов;
* студийные изображения (`default`);
* изображения в реальных условиях (`real_world`);
* агрегирование исходных категорий в 6 целевых классов.

---

## Подготовка данных

После загрузки датасета необходимо выполнить разбиение на обучающую, валидационную и тестовую выборки:

```bash
python src/prepare_dataset.py
```

В результате будет создана структура:

```text
data/
└── processed/
    ├── train/
    ├── val/
    └── test/
```

---

## Обучение моделей

Пример обучения ResNet18:

```bash
python src/train.py --model resnet18
```

Доступные модели:

```text
resnet18
densenet121
mobilenet_v3_large
efficientnet_b0
vit_b_16
```

После обучения сохраняются:

* веса модели;
* история обучения;
* параметры эксперимента.

---

## Оценка модели

Пример оценки:

```bash
python src/evaluate.py ^
    --model efficientnet_b0 ^
    --weights models/efficientnet_b0_best.pth
```

Во время оценки вычисляются:

* Accuracy;
* Precision;
* Recall;
* F1-score;
* Confusion Matrix;
* скорость инференса.

---

## Сравнение моделей

После оценки всех архитектур:

```bash
python src/compare_models.py
```

Будет сформирована итоговая таблица сравнения моделей.

---

## Проверка одного изображения

```bash
python src/inference.py ^
    --image demo/samples/example.jpg
```

Программа выводит:

* класс отходов;
* вероятность;
* рекомендацию по сортировке.

---

## Запуск демонстрационного приложения

```bash
streamlit run demo/app.py
```

После запуска откроется веб-интерфейс, позволяющий загрузить изображение и получить результат классификации.

---

## Основные результаты

По итогам сравнения пяти архитектур лучшей моделью стала **EfficientNet-B0**, показавшая наилучшее сочетание качества классификации и скорости обработки изображений.
